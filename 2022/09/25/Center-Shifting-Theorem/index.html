移心定理及其推广

前言：首先来看原题

灵感来自于九上数学《课时》，P87第11题图2。

$如下图,已知 A,B,C,D 共圆,AB=2,CD=4;AC \bot BD,求圆的半径r.$
$（可拖动下图点A寻找思路）$

当时我看到这道题给出的 $2$ 和 $4$，就立马想到了$1:2:\sqrt{5}$的比例，大概估计是$Rt \triangle$的直角边，而直径作为斜边。

那么直径$d$就应当长$2 \sqrt{5}$，半径$r= \sqrt{5}$。

不过，上面所描述的一切，都是我在脑中放开想象的，并没有依赖任何定理或公式，仓促地写下答案就溜了。

第二次想这道题就是在数学课上了。正好讲到这题，老师可是要找人讲题的，于是我就一直用铅笔连接各种各样的辅助线，算着算着都把余弦定理用上了。（总归不可能是题目超纲吧）

最后呢，当我把四条半径连上后，这道题最简单的解法终于出来了，答案也就是$2 \sqrt{5}$。但途中几经曲折，我还发现了这个四边形一条极其优美的性质，暂且先把它称作移心定理吧。

移心定理——对角线垂直版

前文所谓定理，不过是几句简单就能推出来的算式，请勿高估

既然要证明是$Rt \triangle$，那也就是要证明$\angle AOB + \angle COD = 180^\circ$。有了垂直，似乎很容易做到。

让我们连接$AC$。（请动手在上图用线段连接$AC$）

于是易得$\angle AOB = 2\angle ACB,\angle COD = 2\angle CAD$。

$\because \angle ACB + \angle CAD = 90^\circ$
$\therefore \angle AOB + \angle COD = 180^\circ$

也就能够将$B,O,D$三点旋转至共线，进而证明是$Rt \triangle$。

不过，让我们来看看经过一番推理之后，我们先后发现了什么：
$\because AD \bot BC$
$\therefore \angle AQB+\angle CQD = 180^\circ$

——似乎出现了两个相等的和：
$\angle AOB + \angle COD = \angle AQB + \angle CQD = 180^\circ$

其中$O$为四边形对角线的交点，先叫它对心；$Q$是四边形外接圆的圆心，为外心。而相对的两个“对顶角”的互补性似乎从点$O$转移到了点$Q$，从对心转移到了外心，这也正对应了定理名称——移心定理。

不过题目还没完，如果两条对角线不垂直，移了心又会怎么样呢？

移心定理推广版

此刻我们将$B$点的锁定解除，让角度可以自由变换。

于是可以得到$ \angle AOB + \angle COD = \alpha $，变相的“移心”。

$\because \alpha \neq 90^\circ$
$\therefore B,O,D旋转后不会共线$

此时若想求半径$r$，则必须将$\triangle AOB $绕$O$旋转至$AO$与$CO$重合（记新三角形为$\triangle A’OB’$），再连上$B’D$。

$\because \angle B’OD = 360^\circ -(\angle AOB + \angle COD) = 360^\circ - \alpha$
$\therefore \angle B’A’D = \frac {1}{2} \angle B’OD = 180^\circ - \frac {1}{2} \alpha$

根据余弦定理，易得$B’D^2 = AB^2 + CD^2 - 2AB \cdot CD \cos (180^\circ - \frac {1}{2} \alpha)$

而$2r^2 -2r^2 \cos (360^\circ - \alpha) = B’D^2$

整理一下，$2r^2 -2r^2 \cos \alpha = AB^2 + CD^2 + 2AB \cdot CD \cos \frac {1}{2} \alpha$。

当然如果我们把$r$这一项单提出来的话会得到一个很丑的式子。

总结

所谓移心定理，其实就是将四条以外心为端点的线段，转换成以对心为端点的线段，再去探求它们之间的联系。
